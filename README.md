Git-guide.txt - обновляю по мере изучения git<br><br>
pyLogger.py - универсальный логгер, запускается при помощи .bat файлов с аргументами (какой прибор, параметры измерения и др.).<br>
Использует .ini файлы для настройки состояния прибора.<br><br>
pyRetriever_VAH-meter - ВАХ-метр (задумывалось как универсальная оболочка).<br>
Измерение ВАХ реализовано для Tonghui TH1992B.<br>
Использует Tonghui_TH1992B_config.ini для настройки состояния прибора, а также еще один - sweep_config.ini, для хранения параметров сканирования.<br>
Данные автоматически в файл не пишутся, необходимо сохранять после каждого измерения.<br>
В случае построения нескольких кривых на одном графике сохранена будет только последняя измеренная.<br><br>
Axes_Scan_Tonghui.py - сканирование ножами и измерение TH1992B или TH2690A.<br><br>
Axes_Scan_OnTime.py - сканирование ножами с фиксированным шагом по времени без измерений.<br><br>
Legacy - картирование.<br>
Сканирует ножами, перемещая окошко по карте. Снимает показания при помощи TH1992B или TH2690A.<br>
Рефакторинг под новые библиотеки пока не выполнен.<br><br>
