Git-guide.txt - обновляю по мере изучения git<br><br>
pyLogger.py - универсальный логгер, запускается при помощи .bat файлов с аргументами (какой прибор, параметры измерения и др.).<br>Использует .ini файлы для настройки состояния прибора.<br><br>
pyRetriever.py - ВАХ-метр (задувалось как универсальная оболочка).<br>Измерение ВАХ реализовано для Tonghui TH1992B. Использует .ini файл для настройки состояния TH1992B, а также еще один - sweep_config.ini, для хранения параметров сканирования.<br><br>
